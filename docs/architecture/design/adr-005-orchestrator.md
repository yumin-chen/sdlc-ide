# ADR-005: Orchestrator

## 1. Context

This ADR describes the design of the Orchestrator component, which governs the Core DAG and validates the Mesh layer as defined in ADR-001. This ADR must also detail the Orchestrator's role as the first stage in the two-stage validation pipeline for the autonomous extension system described in ADR-010, including its responsibilities for structural validation and cycle detection.
